# Data Folder

This folder holds the local vector-store artifact produced by the ingestion pipeline.

## `vector-store.json`

Generated by `POST /api/ingest`. Contains a JSON array of `VectorRecord` objects:

```json
[
  {
    "id": "<guid>",
    "source": "Grocery_Store_SOP.md",
    "chunkText": "<section content>",
    "embedding": [0.012, -0.034, ...],
    "metadata": {
      "Index": "0"
    }
  },
  ...
]
```

The file is loaded into memory on every `SearchAsync` call and queried via in-memory cosine similarity. The path is configurable via `Challenge:VectorStorePath` in `appsettings.json` (or `Challenge__VectorStorePath` as an environment variable).

This file is excluded from version control (`.gitignore`). Run ingestion once after first startup to populate it.

## `Grocery_Store_SOP.md` (symlink / copy)

In Docker, the SOP document is copied into this directory at image build time. In local development the `IngestController` resolves the path relative to `ContentRoot` and falls back to `../../../../knowledge-base/Grocery_Store_SOP.md` when the configured path does not exist.
